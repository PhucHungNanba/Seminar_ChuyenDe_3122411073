# 05.1.1 Setup Frontend & Admin Portal (Scaffolding Checklist)

Đây là checklist thực thi việc khởi tạo và cấu hình môi trường Frontend Storefront (Khách hàng) và Admin Portal (Quản trị) sử dụng Vite, React và TypeScript. Hãy thực hiện từng bước và kiểm thử tuần tự theo hiến pháp Vibe Coding.

---

## 📋 Danh sách các đầu việc thực thi (Checklist)

- [x] **Bước 1: Khởi tạo dự án Frontend (Storefront App)**
  - Chạy lệnh khởi tạo Vite React (TypeScript) tại thư mục `Frontend`:
    ```bash
    npm create vite@latest Frontend -- --template react-ts
    ```

- [x] **Bước 2: Khởi tạo dự án Admin Portal (Web Quản trị)**
  - Chạy lệnh khởi tạo Vite React (TypeScript) tại thư mục `Admin`:
    ```bash
    npm create vite@latest Admin -- --template react-ts
    ```

- [x] **Bước 3: Thiết lập Tailwind CSS v4 cho cả hai dự án**
  - Cài đặt Tailwind CSS v4 và cấu hình các plugin hỗ trợ.
  - Tạo/cấu hình tệp `tailwind.config.js` (hoặc `@import "tailwindcss";` trong `index.css` theo chuẩn Tailwind CSS v4) ở cả hai dự án.

- [x] **Bước 4: Cài đặt các thư viện lõi (Core Libraries)**
  - Di chuyển vào từng thư mục và cài đặt:
    ```bash
    npm install zustand react-router-dom lucide-react framer-motion axios
    ```
  - Cài đặt devDependencies cần thiết (nếu có).

---

## 🧪 HOẠT ĐỘNG KIỂM THỬ (Verification Steps)

> [!NOTE]
> ### 🔍 Quy trình Kiểm tra
> 1. Di chuyển vào thư mục `Frontend` và chạy `npm run dev`.
> 2. Di chuyển vào thư mục `Admin` và chạy `npm run dev`.
> 3. Kích hoạt `Browser Subagent` truy cập lần lượt vào các địa chỉ máy chủ phát triển.
>
> ### 🎯 Tiêu chí Pass
> - Cả hai máy chủ phát triển (Dev Server) Vite đều khởi chạy thành công không có lỗi crash.
> - Trình duyệt truy cập hiển thị giao diện mặc định ban đầu của Vite + React + TypeScript một cách hoàn hảo ở cả hai cổng cổng `5173` và `5174`.
